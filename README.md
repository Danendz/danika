# Danika

`Danika` is a Super App that contains implementation of different social media modules like: 

- [ ] Full profile implemntation
- [ ] Calendar scheduler like in Google Calendar
- [ ] Live chat
- [ ] Posts

Frontend: 
- NextJS,
- Zustand,
- Motion/React,
- Shadcn,
- TailwindCSS v4,
- TRPC/Client
- Auth.js

Backend:
- TRPC/Server
- Minio Client
- Zod
- Prisma ORM
- Postgresql

Deploy:
- Railway

#### Work is still in progress!
